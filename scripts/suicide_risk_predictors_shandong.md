## Overview

This analysis explores predictors of the suicide-attempt outcome
(`Died`: yes/no) in the Shandong suicide attempts dataset, and fits a
binomial logistic regression model, with associated diagnostics.

**Variables of interest:** `Died` (outcome); `Urban`, `Age`, `Sex`,
`Education`, `Method` (predictors).

## Data Preparation for Modeling

``` r
df <- data.frame(
  died      = factor(data$Died),
  urban     = factor(data$Urban),
  sex       = factor(data$Sex),
  age       = data$Age,
  education = factor(data$Education),
  method    = factor(data$Method)
)
```

``` r
# Set reference levels
df$urban_RL     <- relevel(df$urban, ref = "unknown")
df$education_RL <- relevel(df$education, ref = "unknown")
df$Method_RL    <- relevel(df$method, ref = "unspecified")
```

## Logistic Regression Model

``` r
fit <- glm(
  died ~ urban_RL + age + education_RL + sex + Method_RL,
  family = binomial,
  data   = df
)
```
