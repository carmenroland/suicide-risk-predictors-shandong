# Suicide Risk Predictors in Shandong, China

## Project Overview

This project investigates sociodemographic predictors associated with suicide mortality in Shandong, China using exploratory data analysis and logistic regression.

The analysis was conducted as part of graduate research at Texas A&M University.

## Research Questions

- Which demographic factors are associated with suicide mortality?
- Which predictors have the strongest statistical relationship with suicide outcomes?
- How well can logistic regression explain variation in suicide risk?

## Methods

- Data cleaning and preprocessing
- Exploratory data analysis
- Logistic regression modeling
- Statistical interpretation of predictors
