# Analysis Outputs

## Purpose

This folder contains figures, visualizations, and model evaluation outputs generated during the analysis of suicide risk factors in Shandong, China.

These outputs were produced from the R Markdown workflow located in the `scripts/` directory and were used to support exploratory data analysis, statistical modeling, and interpretation of results.

## Contents

Outputs include:

### Exploratory Data Analysis (EDA)

Visualizations used to examine the distribution of variables and identify potential patterns within the dataset, including:

- Univariate distributions
- Frequency plots
- Demographic summaries
- Exploratory comparisons between predictor variables

### Bivariate Analysis

Figures used to investigate relationships between predictors and suicide mortality outcomes, including:

- Group comparisons
- Cross-tabulated visualizations

### Logistic Regression Results

Visualizations and summaries generated from logistic regression modeling, including:

- Confidence intervals
- Predictor significance assessments

### Model Evaluation

Diagnostic outputs used to assess model performance and validity, including:

- ROC curves
- Classification performance metrics
- Model fit diagnostics
- Interaction model comparisons

## Reproducibility

All outputs in this directory are generated from the analysis workflow documented in:

```text
scripts/shandong_suicide_analysis.Rmd
```

Users can reproduce these outputs by running the analysis script with the appropriate dataset and package dependencies.
