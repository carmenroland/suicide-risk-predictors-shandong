# Project Documentation

## Purpose

This folder contains the primary written deliverables for the Shandong suicide risk project.

These documents summarize the project's objectives, methodology, statistical analyses, results, and conclusions in formats intended for both technical and non-technical audiences.

## Contents

### Final Research Report

The final project report provides a comprehensive overview of the analysis, including:

- Research background and motivation
- Study objectives and research questions
- Dataset description
- Statistical methodology
- Exploratory data analysis
- Logistic regression modeling
- Model evaluation and diagnostics
- Interpretation of findings
- Discussion of limitations
- Conclusions and recommendations

This document serves as the primary technical summary of the project.

### Project Presentation

The project presentation provides a concise visual summary of the analysis and key findings.

Topics include:

- Project overview
- Data description
- Analytical approach
- Major statistical findings
- Visualizations and model results
- Conclusions and implications

This presentation was designed to communicate the project efficiently to an audience of researchers, instructors, and stakeholders.

## Relationship to Other Project Components

The documents in this folder summarize and interpret the work performed in the repository.

Supporting materials can be found in:

- `../scripts/` – R Markdown analysis workflow and statistical code
- `../output/` – Generated figures, visualizations, and model outputs
- `../data/` – Dataset documentation and source information

## Recommended Reading Order

For visitors exploring the project:

1. Review the root project README for an overview
2. Read the project presentation for a high-level summary
3. Read the final report for detailed methodology and findings
4. Review the analysis scripts for implementation details
